# Yunex Traffic — Branding Cheat Sheet (One Page)

---

## 1. Logo

**Primary logo**
- Wordmark: **Yunex Traffic** (always two words, both capitalised)
- Suitable for **light and dark backgrounds** with sufficient contrast
- **Minimum size**
  - Digital: **110 px**
  - Print: **28 mm**
- **Clear space**: minimum clearance equals the height of the **“YUNEX”** lettering
- **Do not** stretch, rotate, recolour, add shadows, apply effects, or alter proportions

**Micro logo**
- Used only when minimum size cannot be met (e.g. small product labels or giveaways)

---

## 2. Colour Palette (Use HEX Exactly)

### Core Brand Colours
| Colour | HEX | Typical Usage |
|------|-----|---------------|
| Black | `#000000` | Primary text, logos |
| White | `#FFFFFF` | Backgrounds |
| Royal Blue | `#1E2ED9` | Primary highlight, links, emphasis |
| Green | `#00E38C` | Secondary highlight |
| Gray | `#E4EDED` | Neutral panels, subtle backgrounds |

### Secondary / Accent Colours  
*(Use only when the core palette is not sufficient)*

| Colour | HEX |
|------|-----|
| Orange | `#F47738` |
| Yellow | `#FFE564` |
| Purple | `#A483FF` |
| Lavender | `#9DBBFF` |

---

## 3. Gradients (Approved Only)

### Primary Gradient — **Yunex Silver**
```

\#688ABA → #FFFFFF → #DEECFF → #AFFAD7

```

### Secondary Gradients
- **Frosted**: `#9DBBFF → #DEECFF → #FFFFFF`
- **Spring**: `#DEECFF → #E4EDED → #AFFAD7`

### Gradient‑only Colours  
*(Never use these as flat fills)*

- Steel: `#688ABA`
- Sky: `#DEECFF`
- Pistachio: `#AFFAD7`

**Rules**
- Gradients are for **highlights only**
- Transparency and cropping allowed
- Never combine multiple gradients in one asset

---

## 4. Typography

### Corporate Fonts (Preferred)
| Usage | Font | Weight |
|-----|------|--------|
| Headlines | **Jeko** | SemiBold |
| Sub‑headlines | **Jeko** | SemiBold |
| Body text | **Inter** | Regular |
| Emphasis | **Inter** | Medium |
| Quotes / callouts | **Jeko** | Regular |

### System Fonts (Editable Office Documents)
| Usage | Font |
|-----|------|
| Headlines | Arial Bold |
| Body text | Arial Regular |

### Typography Rules
- Left‑aligned text only
- Aim for **≤ 15 words per line**
- Default text colour: **Black**
- Highlight text only with **Royal Blue**
- Use white space deliberately

---

## 5. Visual Style & Imagery

### Photography
- Bright, modern, urban environments
- Real cities and real people
- Avoid:
  - Night photography
  - Empty streets
  - Overly “stock” images
  - Heavy motion blur

### Data Visuals
- Black/white base + **one** highlight colour
- Flat, dashboard‑inspired visuals
- Optional gradient accents

### Icons & Illustrations
- **Icons**: simple black & white line icons
- **Mini illustrations**: minimal detail, brand gradients
- **Large illustrations**:
  - Central focus object
  - Abstract city silhouettes
  - Gradients used sparingly as highlights

---

## 6. Tone & Brand Feel

- Efficient, friendly, confident
- Balance **human benefit** with **technology & facts**
- Market‑leading but approachable
- Clean, structured, uncluttered layouts

---

## 7. Quick Do / Don’t

### ✅ Do
- Use black/white plus **one** accent colour
- Use Jeko and Inter only
- Keep layouts clean and spacious

### ❌ Don’t
- Mix multiple accent colours
- Apply unbranded effects or shadows
- Modify or distort the logo in any way
